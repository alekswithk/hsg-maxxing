# Microeconomics III - Notion Embedding

This repo is used for html files produced by Claude that are being used as helper methods for studying for the exam and understanding the topics.

## Integrating new files in Notion

To integrate new files, create a new `.html` with the corresponding content, push it to github and use the `https://alekswithk.github.io/micro_III_notion/your_file.html` link to embed it into Notion.
